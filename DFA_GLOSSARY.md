### **S-Axis (Structural Axis)**
**Full Name**: Structural Axis of Reality

**Definition**: The component of reality representing:
- Localization (position in space)
- Mass/charge (material properties)
- Constraint (boundaries and limits)
- Persistence (stability over time)
- Individual identity (discrete entities)

**Mathematical Representation**:
- S (scalar quantity)
- Range: 0 ≤ S ≤ 1 (normalized)

**Physical Examples**:
- **Particle Physics**: Rest mass, charge, spin
- **Proteins**: Amino acid sequence, structural domains
- **Cells**: Membrane boundaries, organelles

- **Societies**: Laws, institutions, infrastructure

**Metaphor**: "The skeleton" - provides structure but cannot move alone

**Common Mistakes**:

### **Correlation Integral C(r)**
**Definition**: Mathematical function measuring fraction of point pairs separated by distance less than r.

**Formula**:
```
C(r) = (1/N²) × Σᵢ≠ⱼ Θ(r - |xᵢ - xⱼ|)

Where:
- N = number of points
- xᵢ, xⱼ = positions of points i and j
- Θ = Heaviside step function (1 if true, 0 if false)
- |xᵢ - xⱼ| = distance between points
```

**Power Law**:
```
C(r) ∝ r^D₂

Therefore:
log C(r) = D₂ × log(r) + constant
```

**Measurement**:
1. Compute all pairwise distances
2. For each r, count pairs within distance r
3. Plot log C(r) vs log r
4. Slope = D₂

**Usage**: "The correlation integral scales as C(r) ∝ r^1.46"
